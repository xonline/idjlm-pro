How mappings are used

These mappings control what ID3 frames or export fields each IDJLM-track attribute writes to. The defaults match what the tag writer (app/services/tag_writer.py) produces:

  • Rekordbox & Serato: writes ID3 frames directly — BPM to TBPM, key to TKEY+TXXX:INITIALKEY, etc.
  • M3U: populates EXTINF comment strings with the selected fields.
  • CSV: uses the mapped names as column headers in the exported spreadsheet.
